# Penrose

一个黄黑配色的简洁 Beamer 主题

# 使用

复制 `beamerthemePenrose.sty` 和 `res/` 到工程文件夹即可

```tex
\documentclass[aspectratio=169]{ctexbeamer}
\usetheme{Penrose}
```
